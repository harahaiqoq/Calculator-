🧮 Premium Glassmorphism Calculator

Ek modern, sleek aur high-performance web-based calculator jo **Glassmorphism** aur **Dark Neo-Brutalism** design principles par based hai.

🚀 Live Demo
Aap isse yahan live dekh sakte hain:

## ✨ Features
- **Modern UI:** Glass-morphic design blur effects ke saath.
- **Responsive:** Mobile, Tablet aur Desktop sab par perfect chalta hai.
- **Smart Logic:** Division errors aur long decimals ko automatically handle karta hai.
- **Cyberpunk Vibe:** Dark theme jo aankhon ke liye comfortable hai (Eye-friendly).

## 🛠️ Tech Stack Used
- **HTML5:** Semantic structure ke liye.
- **CSS3:** Custom animations, Flexbox, aur Grid layouts.
- **JavaScript (ES6):** Dynamic calculations aur functional logic.
